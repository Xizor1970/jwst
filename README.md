## What is this?

An interconnected knowledge base exploring the James Webb Space Telescope — its science, engineering, politics, and human story. Built as a learning companion for anyone curious about what JWST is revealing about our universe.

Each page combines:
- **Official NASA/STScI imagery** from JWST's first years of operation
- **Clear explanations** pitched for someone with a technical background but not astronomy expertise
- **Analogies** that make cosmic-scale numbers tangible
- **Unanswered questions** — the frontiers where knowledge ends
- **Speculation sections** (clearly marked) — exploring ideas at the edge of current understanding

## Interactive Animations

Seven Canvas-based visualisations (zero dependencies — vanilla JS):

| Animation | Concept |
|-----------|---------|
| **Redshift** | Watch a UV wave stretch to infrared as z increases from 0 to 15 |
| **Transit** | A planet crosses its star — live light curve + ppm depth calculation |
| **Sunshield** | 5-layer thermal cascade from 383 K to 50 K |
| **Ramp Sampling** | Cosmic ray detection via non-destructive reads + slope fitting |
| **Lensing** | Mouse-controlled lens — arcs and multiple images form in real time |
| **Microshutters** | Full magnet sweep + electrostatic latch actuation cycle |
| **L2 Orbit** | Sun/Earth/JWST orbital mechanics with halo orbit |

Each animation includes a "What you're seeing" + "So what?" explainer.
